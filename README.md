# Magical Odyssey Portfolio (Astro)

Retro-futuristic, celestial-themed portfolio built with Astro + Tailwind.

## Setup

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Included pages

- `/` Home (navigation hub)
- `/projects`
- `/soft-skills`
- `/skills`
- `/experience`
- `/ai-portal`

## Notes

- Data-driven content stored in `src/data/*.json`
- Motion effects are CSS-first with reduced-motion support
- Deployment-ready for Vercel or Netlify static hosting
