# Magical Odyssey Portfolio (Astro)

Retro-futuristic, celestial-themed portfolio built with Astro + Tailwind.

## Setup

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Included pages

- `/` Home (navigation hub)
- `/projects`
- `/soft-skills`
- `/skills`
- `/experience`
- `/ai-portal`

## Project notes

- Data-driven content is in `src/data/*.json`
- Replace placeholder links (`#`) and image paths in `src/data/projects.json`
- Animations include reduced-motion support
- Static output is deployment-ready for Vercel/Netlify
